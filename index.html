Desktop browser

Desktop browsers have evolved a lot over the years, but they still waste the screen space with huge toolbars and elements the user does not always need. This concept - which has recieved many iterations over the years (only partially shown here) - aims to fix that by merging address bar and bookmarks bar with the tabs. Apple has released a new version of Safari that has a very similar design, so this concept is now discontinued.
